# Test_Addressebles

Тестовый Unity-проект, в котором настраивается и демонстрируется работа системы Addressables.

## О проекте

Цель проекта — показать базовую интеграцию **Unity Addressables**:

- разнесение контента по Addressables-группам;
- загрузку объектов по ключам в рантайме;
- работу с вынесенными данными для билда (папка `ServerData/WebGL`).

Это не законченная игра, а технический прототип, собранный как тестовое задание.

## Структура проекта

- `Assets/` — основной игровой и служебный контент проекта.
- `ServerData/WebGL/` — сгенерированные данные Addressables для билда (каталоги и asset bundles).
- `Packages/`, `ProjectSettings/` — стандартные служебные папки Unity.

## Что демонстрирует этот проект

Базовую настройку Addressables в Unity-проекте.
Использование отдельной папки ServerData для данных, которые могут лежать на сервере.
Подход к техническому тестовому: минимальная сцена, фокус на настройке системы загрузки
